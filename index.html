Ola – All You Need in a Teacher's Best Assistant

NLP Computer Vision Deep Learning Transformer Models AI

Ola is an AI-powered educational assistant designed to support teachers by automating key classroom tasks. The project features two core modules: Automated Lecture Note Generation and Whiteboard Text Refinement. The note generation module combines OCR-extracted slide text with audio transcriptions using a fine-tuned Whisper model, enhanced through a T5-based summarization model. The whiteboard module uses a drawing canvas and a deep Capsule Network trained on EMNIST.

  • Automated lecture note generation with OCR and audio transcription
  • Fine-tuned Whisper model for speech-to-text conversion
  • T5-based summarization for content enhancement
  • Deep Capsule Network for handwritten text recognition
